# customer-churn-analysis-powerbi
📊 Análisis de Churn de Clientes | Power BI

📌 Descripción

Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI para analizar la cancelación de clientes (churn) y detectar los principales factores que influyen en la pérdida de usuarios.

El objetivo es transformar datos en insights accionables que permitan mejorar la retención y reducir el impacto económico del churn.

🎯 Objetivos del análisis

Medir la tasa de cancelación de clientes
Identificar segmentos con mayor riesgo de churn
Analizar el impacto económico de la pérdida de clientes
Detectar oportunidades de mejora en la retención

🛠️ Herramientas utilizadas

Power BI
Modelado de datos
DAX (medidas y KPIs)

📊 KPIs principales

Tasa de cancelación (% churn)
Cantidad de clientes
Clientes perdidos
Ingresos perdidos

🔍 Insights clave

Los clientes con contrato mensual presentan la mayor tasa de churn
La cancelación es significativamente más alta durante los primeros meses
Los clientes con menos servicios contratados tienen mayor probabilidad de abandono
Los pagos automáticos están asociados a menor tasa de churn

💡 Valor del análisis

Este dashboard permite:

Comprender el comportamiento de los clientes
Detectar factores críticos de cancelación
Priorizar estrategias de retención
Cuantificar el impacto económico del churn
